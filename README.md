# Interactive Personal Blog Platform

A simple, client-side blog application that lets users create, edit, and delete blog posts — all stored locally in the browser using `localStorage`.

# How to Run

1. Download or clone the repository  
2. Open `index.html` in any modern web browser  
3. Start writing and managing your blog posts!

# Features

- Add new blog posts with title, content, and optional image
- Edit and delete existing posts
- Form validation with helpful error messages
- Automatically tracks when a post was edited
- Clean, responsive layout with image support
- Data is saved in `localStorage` — no backend needed
